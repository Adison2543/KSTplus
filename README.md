# newKST
 
install project as :
- `composer install`
- `npm i`
- Create Database name `kstid`
- `php artisan migrate`
- `php artisan db:seed`

to start dev server is:
- `npm run dev`
- `php artisan serve`

Don't forget to update the .env file after pulling the project from Git.

after you install project,the default admin user is : 
- Username: `admin`
- Password: `iddrivesadmin`
